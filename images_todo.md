# 画像化バックログ（図の抽出タスク）

図があると解きやすい問題のうち、まだ `images/{id}.png` を用意できていないものの一覧。
`pdf/extracted/excluded.md` が図依存問題の年度別カタログ（stem図／選択肢が図 の区別つき）。

## 抽出手順（前回踏襲）
1. `pdftoppm -jpeg -r 62 -f N -l N pdf/{年度}_qs.pdf /tmp/pc/loc` でページ特定（印刷ページ番号 = PDFページ番号, offset=0）。問密度は約2〜3問/ページ（分野により変動、ストラテジ系は粗い）。
2. `pdftoppm -png -r 300 -f N -l N pdf/{年度}_qs.pdf /tmp/pc/hi{id}` で高解像度化（約2150×3040px）。
3. PIL で図領域を crop → `.convert('L')` → 幅1000px超なら LANCZOS 縮小 → `optimize=True` で `images/{id}.png` 保存。
4. `data/questions.json` の該当 id に `"image":"images/{id}.png"` を追加。図の文章代替説明があれば除去（**問題文末尾の「〜はどれか。」を消さないよう注意**）。
5. まとめて `npm run build`、`sw.js` の `CACHE` を bump。
