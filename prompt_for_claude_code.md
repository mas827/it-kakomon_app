## 作業手順（1問ずつ繰り返す）

### Step 0（初回のみ）：アプリ側に画像表示コードを追加

`fe-study-app.jsx` の `SessionScreen` 内、問題文表示の直後に以下を追加する。
**これは全問通じて1回だけ行う。**

```jsx
{/* 既存コード（問題文） */}
<p style={{ fontSize: 17, lineHeight: 1.7, fontWeight: 500, margin: "0 0 24px", whiteSpace: "pre-wrap" }}>
  {q.question}
</p>

{/* ↓ここを追加（問題文の直後、選択肢の直前） */}
{q.image && (
  <img
    src={q.image}
    alt="問題の図"
    style={{
      width: "100%",
      maxWidth: 420,
      borderRadius: 8,
      border: `1px solid ${C.border}`,
      margin: "-8px 0 24px",
      display: "block",
      backgroundColor: "#fff"
    }}
  />
)}
```

### Step 1：問題PDFの該当ページを画像化

```bash
# 例：令和元年度秋期 問1（p.3）の場合
pdftoppm -jpeg -r 200 -f 3 -l 3 pdfs/2019r01a_fe_am_qs.pdf /tmp/page

# 生成されたファイル名を確認（ページ数によってゼロパディングが変わる）
ls /tmp/page-*.jpg
```

**ページ番号の特定方法：**
PDFのページ番号（ファイル先頭からの通し番号）と問題冊子のページ番号（印刷されている番号）は
表紙・注意事項ページ分だけズレる。上記の対応表のページ数を起点に、前後ページも確認すること。

---

### Step 2：ページ画像を確認して問題内容を把握

生成した画像を確認し、以下を読み取る：
- **問題文**（テキスト部分）
- **図・グラフ・回路図**の内容と範囲
- **選択肢**（テキストまたは図）
- 図が問題文に含まれるか、選択肢が図であるかを判断

または解答PDFを直接確認する。正解の記号（ア〜エ）を上記インデックス表で数値に変換する。

### Step 4：図をトリミングして images/ フォルダに保存

```python
from PIL import Image

img = Image.open("/tmp/page-003.jpg")

# 図の領域をトリミング（ページ全体から図の部分のみを切り出す）
# 座標は左(left), 上(top), 右(right), 下(bottom) の順
# 実際の座標は画像を確認して調整すること
cropped = img.crop((left, top, right, bottom))

# images/ フォルダがなければ作成
import os
os.makedirs("images", exist_ok=True)

cropped.save("images/NO1-01.png")
```

**トリミングの方針：**

| 図の種類 | 切り出す範囲 |
|----------|------------|
| 問題文中の図のみ（流れ図・タイミング図など） | 図の部分のみ |
| 選択肢が図（グラフ・回路図など） | 選択肢ア〜エの図をまとめて1枚 |
| 問題文に図＋選択肢にも図 | 問題文の図と選択肢の図を合わせて1枚 |

**選択肢が図の場合の choices の書き方：**
```js
choices: ["ア", "イ", "ウ", "エ"],  // テキストは「ア〜エ」のみ
// → image に選択肢ア〜エの図をまとめた画像を入れる
```

## 注意事項

**正解番号について**
- 解答PDFの「ア・イ・ウ・エ」→ JSの `answer` は `0・1・2・3`
- 必ず解答PDFで確認すること（推測で入力しない）

**問題文の入力について**
- 問題文は **正確に** 入力すること
- 数式や記号（←、≧、÷ など）はPDFから正確にコピー
- 改行は問題の意味が変わらない範囲で整理してよい

**解説について**
- 解答PDFに解説がある場合はそれをベースにする
- 解答PDFに解説がない場合（正解のみ記載）は、問題文・図・正解から根拠を簡潔に記述する
- 「以下の真理値表参照」のような図への言及は書かない

**QUESTIONS配列への挿入位置**
- `id` 順（NO1-01 → NO1-03 → NO1-05 → ...）を維持すること
- 同じ年度の問題は問番号順に並べる

**ビルドと動作確認**
- 数問追加するたびに `npm run build` を実行してエラーがないか確認する
- `images/` フォルダのファイルもGitHubにアップする必要がある
